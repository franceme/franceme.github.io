Welcome to FRAC (Frantz's Rule Analysis Checker)!

by Miles Frantz

This is a very rudimentary Python Playground that allows you to analyse your code for common security issues.

Why did I do this? Simply because I am a Ph.D. Security Researcher interested in both Static Code Analysis and I want security tools to be more accessible!

The following Python Security Tools have been included within this playground:

Cryptolation - A self-developed state-of-the-art Python security tool, --currently-- being integrated. Bandit - A Python security tool designed and created by the fine team PyCQA. DLint - A security-based Python linter that extends flake8.

How do you use this?

1) Write your Python code into the text editor or add it as a url (here?code=import os,sys,hashlib;print(hashlib.sha1('This is insecure'))). 2) Wait for the first button on the left to change to "Scan the Code" (loading pyodide). 3) Once you click it, wait for the middle button to change to "View Results". 4) Once you see the results, you can refresh the page to start over or click the "Reset the page" button.
